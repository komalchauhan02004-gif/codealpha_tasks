# Data Analysis & Sentiment Analysis Project

## Overview
This project demonstrates an end-to-end data analytics workflow using Python. It includes data exploration, visualization, pattern identification, and basic sentiment analysis on textual data.

## Objectives
- Perform exploratory data analysis (EDA) to understand data structure and trends  
- Generate meaningful insights using aggregation and summary statistics  
- Apply basic NLP techniques to analyze customer sentiment  
- Visualize results for better interpretation  

## Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- TextBlob  
- Jupyter Notebook  

## Project Structure
- Data loading and preprocessing  
- Exploratory Data Analysis (EDA)  
- Pattern and trend analysis  
- Sentiment analysis on text data  
- Data visualization  
- Conclusion and insights  

## Key Insights
- Identified patterns and trends through category-wise analysis and visualizations  
- Classified text data into Positive, Negative, and Neutral sentiments  
- Visualized sentiment distribution to understand overall customer perception  

## Conclusion
This project highlights the practical application of data analytics and basic NLP techniques to extract insights from structured and unstructured data. It reflects real-world analytical thinking and data-driven decision-making
